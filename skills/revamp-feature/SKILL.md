Scan the codebase for the feature you want to revamp. Identify the key components and dependencies of the existing feature.

Interview me about every aspect of this existing features user want to revamp until we reach a shared understanding. Walk down each branch of the design tree, resolving dependencies between decisions one-by-one. For each question, provide your recommended answer.

Ask the questions at most 5 questions at a time.

If a question can be answered by exploring the codebase, explore the codebase instead.

Use question or questionaire tools.
